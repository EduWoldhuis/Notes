
### OS information
```
/etc/passwd, /etc/group      // User information, group information
/etc/sudoers                 // Sudo information
/etc/init.d/                 // Directory of start-up files

```

### Log files
```
/var/log/auth.log            // Authentication log (things done as sudo)  IMPORTANT: use "last -f" for proper output
/var/log/wmtmp               // Login information (/var/log/btmp is failed login info)
```
